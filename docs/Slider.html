Slider

Interactive horizontal slider UI component for adjusting normalized values (0–1). Can be used for volume, brightness, or other adjustable settings.

Constructor

new Slider(scene, x, y, widthopt, heightopt, initialValueopt)

Source:
  • UI/elements/slider.js, line 21
Example
const slider = new Slider(this, 400, 300, 250, 20, 0.3)
    .on("slider-changed", (value) => console.log("Slider value:", value));
Parameters:
Name Type Attributes Default Description
scene Phaser.Scene The Phaser scene this slider belongs to.
x number The x-coordinate of the slider's position.
y number The y-coordinate of the slider's position.
width number <optional>
 200 The width of the slider track.
height number <optional>
 20 The height of the slider.
initialValue number <optional>
 0.5 The initial value of the slider (from 0 to 1).
