Button

A simple interactive text-based button component. Displays a label and emits events when hovered or clicked.

Constructor

new Button(scene, x, y, text, callback)

Source:
  • UI/elements/button.js, line 21
Example
const startButton = new Button(this, 400, 300, "Start Game", () => {
    console.log("Button clicked!");
}).on("button-clicked", () => console.log("Custom event fired!"));
Parameters:
Name Type Description
scene Phaser.Scene The scene this button belongs to.
x number The x-coordinate of the button's position.
y number The y-coordinate of the button's position.
text string The text displayed on the button.
callback function Function to call when the button is clicked.
